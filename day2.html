Lecture Notes

Day 2 builds on the previous session of Introduction to Web Development and covers detailed HTML concepts.

  1. The concept of Hyperlinks
    Hyperlinks are clickable text or images that navigate users to another page or resource on the web.
  2. What is a markup language ?
    A markup language, like HTML, uses tags to structure and format content on the web.
  3. Installing & Setting up a code editor
    Code editors like VS Code or Sublime allow you to write, edit, and manage HTML files efficiently.
  4. Creating your first webpage
    This involves writing basic HTML structure with tags like <html>, <head>, and <body> to display content.
  5. Understanding tags
    Tags are the building blocks of HTML that define elements such as headings, paragraphs, and links. Example: <h1>, <p>
  6. Types of heading tags
    HTML provides six heading tags: <h1> to <h6> to organize content hierarchically.
  7. Paragraph tags
    Paragraph tags like <p> are used to display blocks of text on a webpage.
  8. Horizontal rule
    The <hr> tag creates a horizontal line to separate content visually.
  9. Line break
    The <br> tag inserts a line break without starting a new paragraph.
  10. Types of Lists
    HTML supports ordered <ol> and unordered <ul> lists to organize content.
  11. Anchor tag
    The <a> tag creates hyperlinks to other pages, websites, or sections within the same page.
  12. Adding images
    The <img> tag is used to embed images into webpages using a source URL or file path.
  13. Mini project
    A small hands-on project to practice creating a complete webpage with text, links, and images.
  14. Wrapping up the video
    Concluding the lesson by summarizing key points and next steps for practicing HTML.
