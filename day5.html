Lecture Notes

Here are the main topics covered in Day 5:

  1. Introduction
    Overview of HTML forms and their purpose in collecting user input.
  2. What is an HTML Form ?
    An HTML form is a section of a webpage used to gather information from users.
  3. Code Section
    Section where HTML form elements are written and demonstrated.
  4. form Tag
    The <form> tag defines the beginning and end of an HTML form.
  5. input Tag
    The <input> tag creates interactive fields like text, checkboxes, radio buttons, and more.
  6. label Tag
    The <label> tag defines a label for an input element to improve accessibility.
  7. Detailed Explanation of Form Structure
    Describes how form elements are organized and structured for better usability.
  8. Name & Value Attributes
    Attributes used to identify form data (name) and assign default or user-selected values (value).
  9. Using name Attribute
    The name attribute allows data to be sent to the server under a specific identifier.
  10. Radio Buttons
    Radio buttons allow users to select only one option from a set of choices.
  11. for Attribute
    The for attribute links a <label> to a specific <input> element by its id.
  12. Setting Radio Button Values
    Values are assigned to radio buttons to differentiate user selections in form submission.
  13. Checkboxes
    Checkboxes allow users to select one or multiple options independently.
  14. Mini Project
    A small project to practice creating forms with various input types.
  15. Placeholder Attribute
    The placeholder attribute displays a hint inside an input field before the user types.
  16. input type="date"
    Creates a field for selecting dates with a date picker interface.
  17. input type="file"
    Allows users to upload files from their device to the server.
  18. input type="email"
    Creates a field that validates email addresses entered by the user.
  19. input type="tel"
    Creates a field for entering telephone numbers with basic input validation.
  20. textarea Tag
    The <textarea> tag creates a multi-line text input field for longer user input.
  21. select Tag
    The <select> tag creates a dropdown menu for users to choose from options.
  22. option Tag
    The <option> tag defines individual choices within a dropdown menu.
  23. button Tag
    The <button> tag creates clickable buttons to submit forms or trigger actions.
  24. Homework
    Practice creating forms with different input types, labels, and buttons.
  25. required Attribute
    The required attribute ensures that users cannot submit a form without filling in the field.
  26. Wrapping Up
    Concludes the lesson by summarizing HTML form elements and their attributes.

Sample Registration Form Code



<form action="#" method="post">
    <label for="username">Username:</label><br>
    <input type="text" id="username" name="username" placeholder="Enter your username" required><br><br>

    <label for="email">Email:</label><br>
    <input type="email" id="email" name="email" placeholder="Enter your email" required><br><br>

    <label for="phone">Phone:</label><br>
    <input type="tel" id="phone" name="phone" placeholder="Enter your phone number"><br><br>

    <label for="dob">Date of Birth:</label><br>
    <input type="date" id="dob" name="dob"><br><br>

    <label>Gender:</label><br>
    <input type="radio" id="male" name="gender" value="male">
    <label for="male">Male</label>
    <input type="radio" id="female" name="gender" value="female">
    <label for="female">Female</label><br><br>

    <label>Hobbies:</label><br>
    <input type="checkbox" id="reading" name="hobbies" value="reading">
    <label for="reading">Reading</label>
    <input type="checkbox" id="gaming" name="hobbies" value="gaming">
    <label for="gaming">Gaming</label><br><br>

    <label for="bio">Bio:</label><br>
    <textarea id="bio" name="bio" placeholder="Write something about yourself"></textarea><br><br>

    <label for="country">Country:</label><br>
    <select id="country" name="country">
        <option value="india">India</option>
        <option value="usa">USA</option>
        <option value="uk">UK</option>
    </select><br><br>

    <button type="submit">Submit</button>
</form>
