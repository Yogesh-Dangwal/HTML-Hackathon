Day 3: List and table

Working with Lists, Tables, and Attributes in HTML

Lecture Video

Lecture Notes

Day 3 dives into HTML lists and tables with multiple examples and best practices.

  1. Introduction
    This section introduces lists and tables in HTML and their basic usage.
  2. List Recap
    A quick review of ordered (<ol>) and unordered (<ul>) lists.
  3. Nested List (Example 1)
    Demonstrates a list inside another list for hierarchical structure like <ul> inside <li>.
  4. Nested List (Example 2)
    Another example showing sub-items within a main list item using <ol> or <ul>.
  5. Nested List (Example 3)
    Shows multiple levels of nested lists for complex data.
  6. Nested List (Example 4)
    Illustrates combining ordered and unordered lists together.
  7. List with Heading
    Lists can include headings (<h1> to <h6>) to organize different sections of content.
  8. Homework – Practice creating lists
    Create your own lists to reinforce understanding of nested and normal lists.
  9. Introduction to HTML Tables
    Tables organize data into rows (<tr>) and columns (<td>) for better readability.
  10. Rows and Columns
    Rows (<tr>) contain columns (<td>), forming the structure of a table.
  11. Table Example 1
    A basic table with a few rows and columns: <table><tr><td>Data</td></tr></table>
  12. Material Icon Theme Extension
    Optional VS Code extension for adding icons while editing HTML files.
  13. table Tag
    The <table> tag creates a table container for rows and cells.
  14. td (Table Data Cell)
    The <td> tag defines a standard cell inside a table row.
  15. Adding Borders to a Table
    Borders can be added using CSS or the border attribute to separate cells visually.
  16. tr (Table Row)
    The <tr> tag groups table cells into a row.
  17. Table Example 2
    Example showing multiple rows and columns with basic borders: <table border="1">...
  18. th (Table Header Cell)
    The <th> tag creates header cells that are bold and centered.
  19. Table Header, Body, and Footer (<thead>, <tbody>, <tfoot>)
    Sections of a table to separate headings, main content, and footer rows.
  20. Quick Review of Tables
    Recap of table tags, rows, columns, and sections for clarity.
  21. Table Example 3
    A table showing the use of <thead>, <tbody>, and <tfoot>.
  22. Understanding Colspan
    The colspan attribute merges multiple columns into a single cell: <td colspan="2">.
  23. Table Example 4
    Example showing how to merge columns using colspan in a table.
  24. Understanding Rowspan
    The rowspan attribute merges multiple rows into a single cell: <td rowspan="2">.
  25. Table Example 5 (Advanced)
    An advanced table demonstrating both colspan and rowspan usage.
  26. Homework – Build sample tables
    Practice creating tables with different rows, columns, colspan, and rowspan.
  27. Wrapping Up
    Concluding the lesson by summarizing lists and tables and their key concepts.
