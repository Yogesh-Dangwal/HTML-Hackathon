Day 4: File Path, Boilerplate, div, class & id

Understanding folder structure, HTML boilerplate, and structure with div, class, id

Lecture Video

Lecture Notes

In Day 4, the focus is on how to organize HTML files, use boilerplate code, and structure content using div, class, and id attributes.

  1. Introduction
  2. What is a File Path – absolute vs relative
  3. HTML Boilerplate Code – what minimum tags every page needs
  4. Understanding folders & file structure
  5. div Tag – a container for grouping elements
  6. class Attribute – applying styles/identifiers to multiple elements
  7. id Attribute – unique identifiers for single elements
  8. Using class & id together
  9. Best Practices for naming classes/ids
  10. Homework Examples

Example Code

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Sample Boilerplate</title>
</head>
<body>
    <div class="container">
        <h1 id="main-heading">Hello World</h1>
        <p class="intro">This is a sample paragraph using class</p>
    </div>
</body>
</html>
