Lecture Notes

Here are the topics covered in Day 1 of the course, explained step by step:

  1. Introduction to the Web Development Course
    Overview of internet basics, networking, and web development concepts.
  2. What is the Internet?
    A global network that connects millions of devices for communication and data exchange.
  3. How is data transferred?
    Information moves in small packets using protocols like TCP/IP.
  4. What is IP Address?
    A unique numerical identifier assigned to every device on a network.
  5. How to access a website
    Entering a URL triggers DNS resolution and loads the webpage in the browser.
  6. Who assigns the IP Address?
    Internet Service Providers (ISPs) assign IPs dynamically using DHCP.
  7. Concept of DNS
    DNS converts human-readable domain names into machine-readable IP addresses.
  8. In-depth Working of DNS
    DNS uses resolver, root, TLD, and authoritative servers to fetch the correct IP.
  9. Problems in Data Transfer and MAC Address
    Data loss, latency, and errors occur; MAC ensures unique identification within networks.
  10. What is Port Numbers?
    Logical communication endpoints used by protocols (e.g., HTTP on port 80).
  11. IPv4 & IPv6
    IPv4 uses 32-bit addresses; IPv6 uses 128-bit for a larger address space.
  12. MAC Address Format
    A 48-bit unique hardware identifier written in hexadecimal (e.g., 00:1A:2B...).
  13. Port Number Format
    Ports are 16-bit numbers ranging from 0 to 65535.
  14. Local Area Network, Switch & Router
    LAN connects nearby devices; switches connect devices; routers connect networks.
  15. Public and Private Addresses
    Public IPs are internet-facing; private IPs are used inside local networks.
  16. Modern Routers
    Provide routing, switching, Wi-Fi, and firewall functionalities in one device.
  17. What is a VPN?
    A Virtual Private Network encrypts traffic and masks IP for secure browsing.
  18. What is Web Development?
    The process of building websites and applications for the internet.
  19. What is the MERN Stack?
    A popular web stack combining MongoDB, Express, React, and Node.js.
  20. Who is a Full-Stack Developer?
    A developer skilled in both frontend and backend technologies.
  21. Wrapping Up the Video
    Summary of networking fundamentals and introduction to web development.

How the Internet Works (Simplified)

  • User types a URL in the browser.
  • Browser sends a request to DNS to resolve domain.
  • DNS returns IP address of server.
  • Browser sends request to that IP.
  • Server responds with HTML/CSS/JS files.
  • Browser renders the website for user.
